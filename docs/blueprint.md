# **App Name**: HighPoint HouseKeep

## Core Features:

- User Authentication and Roles: Firebase Authentication for secure login with role-based access control (Admin, Supervisor, Housekeeper).
- Daily Task Management: Auto-generated daily cleaning assignments based on room occupancy, with real-time task claiming/completion and photo upload for verification.
- Deep Cleaning Scheduler: Configurable rotation calendar for quarterly/annual deep cleans, with automatic alerts and historical completion tracking.
- Inventory Management: Real-time supply level tracking with automated reorder alerts and equipment maintenance logs.
- Performance Dashboard: Individual and team metrics with trend analysis and gamification elements to boost staff performance.
- State Audit Compliance: Pre-formatted reports matching Illinois IDPH Long-Term Care requirements with one-click export and mandatory fields validation.
- Supervisor Tools: Tool that provides a real-time view of staff location/assignments and schedule override capabilities.

## Style Guidelines:

- Primary color: Gold (#D4AF37) to reflect the High Point Residence logo and provide a sense of luxury and cleanliness.
- Background color: Very light desaturated gold (#FAF5E7) to provide a calm, clean backdrop that keeps the gold as the focus.
- Accent color: A darker yellow (#D4A337) will bring users' eyes to important UI elements without clashing with the rest of the scheme.
- Font: 'Open Sans' (sans-serif) for a clean, accessible, and professional look, as requested.
- Note: currently only Google Fonts are supported.
